The Code for LOOPS! 

                        
                            // Get the values from the page
// Starts or Controller Function (initiates the logic and display functions)
function getValues() {
    // Get values from the page
    let startValue = document.getElementById("startValue").value;
    let endValue = document.getElementById("endValue").value;

    // Parse into Integers
    startValue = parseInt(startValue);
    endValue = parseInt(endValue);

    // Validate the inputs to only be integers
    if (Number.isInteger(startValue) && Number.isInteger(endValue)) {

        // Call generateNumbers
        let numbers = generateNumbers(startValue, endValue);
        // Call displayNumbers
        displayNumbers(numbers);

    } else {
        alert("You must enter integers");
    }

}

// Generate numbers from the start value to the end value
// Logic Function(s)
function generateNumbers(sValue, eValue) {
    // Creates an empty array
    let numbers = [];

    // Get all numbers from start value to end value
    for (let index = sValue; index <= eValue; index++) {
        // Will execute the loop until index = eValue
        numbers.push(index);
    }

    return numbers;
}

// Display the numbers and make the even numbers bold
// Display or view Function(s)
function displayNumbers(num) {

    let templateRows = "";
    for (let index = 0; index < num.length; index++) {

        let number = num[index];
        let className = "even";

        // Checks if number is even and makes it bold if it is
        if (number % 2 == 0 && number !== 0) {
            className = "even";
        } else {
            className = "odd";
        }
        templateRows += `<tr><td class="${className}">${number}</td></tr>`;
    }

    document.getElementById("results").innerHTML = templateRows;
}

Function "getValues"

getValues is a control function that executes by adding an addEventListener function to the button. This function grabs the values of the input boxes and stores them in their own variables (startValue and endValue). The two variables are then parsed in attempt to change the value into an integer. Next, the variables startValue and endValue are validated through an if statment to ensure that the value is indeed an integer, otherwise an alert is executed to inform the user that they must enter an integer. If two integers were added, the generateNumbers function is called, then the displayNumbers function.

Function "generateNumbers"

generateNumbers is a function that takes in two arguments and uses them in a for loop to generate an array of numbers that starts at the value indicated in the first input box, and ends at the value indicated in the last input box. For example, if the first input is 5 and the second input is 10, an array of numbers will be generated as such: [5,6,7,8,9,10].

Function "displayNumbers"

The displayNumbers function takes a single argument of type array and adds each item of the array to an html table. In order to determine which item to make bold, an if statment is used to set a variable className to either odd or even. className is added to each table row and a css class is added to the customer css file. If className is "even" then the content of the row will be bold. If className is odd, the content of the row will remained unchanged.
